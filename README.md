# AI 创意绘图工作台

基于 Gitee AI (z-image-turbo) 的文生图 Web 应用程序。

## 功能特性

- 🎨 **AI 图像生成**: 使用 Gitee AI 的 z-image-turbo 模型生成高质量图像
- 📝 **智能提示词**: 支持正向和负向提示词，精确控制生成效果
- ⚙️ **参数调节**: 可调节尺寸、步数、种子等高级参数
- 🔑 **多 API Key**: 在“API 设置”里管理多个 Key，快速切换
- 🖼️ **图像预览**: 支持查看与下载当前生成的图片
- 💾 **一键下载**: 便捷的图片下载功能
- 🌙 **暗色主题**: 现代化的深色 UI 设计
- 📱 **响应式设计**: 支持桌面和移动设备

## 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

应用将在 `http://127.0.0.1:3000` 上运行。

### 构建生产版本
```bash
npm run build
npm run preview
```

## 使用指南

### 1. 配置 API Key
- 点击右上角的 "API 设置" 按钮
- 在列表中添加多个 API Key（可为每个 Key 起名）
- 选择“使用”以设为当前生效的 Key；支持删除
- Key 信息保存在浏览器本地存储（不会上传）

### 2. 设置提示词
- 在 "正向提示词" 区域输入你想要生成的图像描述
- 可选：在 "negative_prompt" 中输入不希望出现的元素

### 3. 调整参数
- **尺寸**: 选择预设尺寸或自定义宽高
- **推理步数**: 控制生成质量和速度
- **种子**: 设置随机种子以复现相同结果

### 4. 生成图像
- 点击 "立即生成" 按钮
- 等待生成完成（可能需要几秒到几分钟）

### 5. 查看和下载
- 生成的图片会显示在右侧画布中
  
- 点击 "保存图片" 下载当前显示的图像

## 技术栈

- **React 18** - 前端框架
- **Vite** - 构建工具
- **Tailwind CSS** - 样式框架
- **Lucide React** - 图标库
- **Gitee AI API** - 图像生成服务

## 注意事项

- 需要有效的 Gitee AI API Key
- 图像生成可能需要几秒到几分钟的时间
- 建议使用详细的中文或英文提示词以获得更好效果
- 图片生成受到 API 速率限制，请勿频繁请求
